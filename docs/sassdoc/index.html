    medya

    @mixin medya($bitis: null, $baslangic: null, $max-width: null, $min-width: null, $medya-query) { ... }

    Description

    Css @media sorgularının süpergücü.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $bitis

    Girilen max-width değeri

    		Numbernull
    $baslangic

    Girilen min-width değeri

    		Numbernull
    $max-width

    Atanan max-width değeri

    		Numbernull
    $min-width

    Atanan min-width değeri

    		Numbernull
    $medya-query

    Medya sorgusu

    		String— none

    Example

    Basit kullanım

    .foo {
  @include medya(xl) {
    color: white;
  }
}
@media screen and (max-width: 75em) { .foo { color: white; }

    Özel değer ile kullanım

    .foo {
  @include medya(1400px) {
    color: white;
  }
}
@media screen and (max-width: 1400px) { .foo { color: white; }

    Unitless değer ile kullanım

    .foo {
  @include medya(1400) {
    color: white;
  }
}
@media screen and (max-width: 1400px) { .foo { color: white; }

    px2em fonksiyon ile kullanımı

    .foo {
  @include medya(px2em(1400px)) {
    color: white;
  }
}
@media screen and (max-width: 87.5em) { .foo { color: white; }

    min-width query oluşturma

    .foo {
  @include min-medya(xl) {
    color: white;
  }
}
@media screen and (min-width: 75em) { .foo { color: white; }

    min-width query oluşturma

    .foo {
  @include medya($baslangic: xl) {
    color: white;
  }
}
@media screen and (min-width: 75em) { .foo { color: white; }

    max-width query oluşturma

    .foo {
  @include max-medya(xl) {
    color: white;
  }
}
@media screen and (max-width: 75em) { .foo { color: white; }

    Aralık belirleme

    .foo {
  $bitis: ul;
  $baslangic: xl;
  @include medya($bitis, $baslangic) {
    color: white;
  }
}
@media screen and (min-width: 75em) and (max-width: 87.5em) { .foo { color: white; }

    Aralık belirleme

    .foo {
  @include medya($baslangic: xl, $bitis: ul) {
    color: white;
  }
}
@media screen and (min-width: 75em) and (max-width: 87.5em) { .foo { color: white; }

    Aralık belirleme

    .foo{
  $bitis: ul;
  $baslangic: xl;
  @include aralik-medya($baslangic, $bitis){
    color: white;
  }
}
@media screen and (min-width: 75em) and (max-width: 87.5em) { .foo { color: white; }
